#DATA 221 ASSIGNMENT 1
this repository contains my solutions for Assignment 1 of DATA 221, each question is in a seperate python file as required.

## Files Overview

File 1: This is a controlled multiplication loop that multiplies consecutive integers starting from 1 until the product exceeds a given threshold

File 2: Defines a fucntion that takes a list of strings returns a nested dictionary containing each string's length and parity

File 3: Defines a power function and safely applies it to a list of (x, y) pairs, skipping pairs with negative exponents.

File 4: Generates random values, sorts them, and finds indices where values are greater than or equal to a randomly generated number.

File 5: Defines a function that validates circle radii, computes circle areas, and returns the percentage coverage of the smaller circle relative to the larger one.
